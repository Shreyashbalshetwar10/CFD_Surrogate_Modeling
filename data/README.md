# Dataset

This project uses the **DeepCFD** dataset for training and evaluating neural operator-based surrogate models for Computational Fluid Dynamics (CFD).

The dataset is **not included** in this repository.

## Dataset Source

DeepCFD Repository:
https://github.com/mdribeiro/DeepCFD

Please follow the instructions provided in the original repository to obtain the dataset.

## Notes

* This repository only contains the implementation, model architectures, and evaluation pipeline.
* All credit for the dataset belongs to the original DeepCFD authors.
